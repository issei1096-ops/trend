# プレスリリース作成・確認フロー標準化マニュアル

**【目的】**
プレスリリース原稿作成の属人化を防ぎ、情報の抜け漏れやブランド毀損リスクを排除する。外部委託（ぽぽちゃん）等と連携した「確実なファクトチェック」の標準フローを確立する。

---

## 1. 原稿作成（社内 0→1）
- **担当**: PR担当者（広報部）
- **作業内容**: 新商品、アライアンス、定期便リニューアル等の要件骨子をもとに、以下の要素を一つのドキュメント（Google Docs等）にまとめる。
  - **背景・想い**: 「たった一杯で幸せになる」ブランド文脈に沿っているか。
  - **商品/サービス情報**: 価格（税抜/税込表記）、販売期間、対象店舗。
  - **素材提供**: メディアが扱いやすい高解像度の画像素材、AI（Gemini等）で生成・調整した指定キービジュアル。

## 2. ファクトチェック（外部委託チェック）
- **担当**: ぽぽちゃん（業務委託）および 製品・ブランド管理部連携
- **チェック項目**:
  - **用語の統一**: 「猿田彦珈琲」「オキーニョ」「eギフト」などの表記揺れ、誤字脱字。
  - **ファクトの裏付け**: アレルギー情報、賞味期限の事実確認（リスク管理事項）。
  - **価格・仕様の正確性**: （例：新定期便の「送料無料・80円末尾ルール」と矛盾していないか等）。
  - **トーン＆マナー**: 「安売り」や「割引」といったブランド価値を下げる表現が含まれていないか。

## 3. 広報内確定＆社長承認ルート
- **担当**: PRマネージャー → 大塚社長
- **作業内容**: 
  1. ファクトチェックを通過した原稿をPRマネージャーが最終確認。
  2. メディア等での見え方（炎上・コンプライアンスリスク等）をスクリーニング。
  3. 大塚社長へ（LINEやChatwork等で）直接根回し・承認を得る。（※独断でのメディア対応・リリース配信は厳禁というリスク管理ルールに従う）
  4. 配信準備（PR TIMES等へのセット）および メディア担当者への持参・直接ヒアリングの準備。
